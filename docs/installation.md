## Detailed iPad instructions

### Download methods

**Safari (easiest)**
1. Open the download link in Safari
2. Tap the download arrow (top right) when prompted
3. Tap the file in Downloads → Open in Minecraft

**AirDrop from Mac**
1. Download `.mcaddon` on Mac from the link above
2. AirDrop to iPad
3. Tap → Open in Minecraft

**Files / iCloud**
1. Save to iCloud Drive or Dropbox
2. Open **Files** app on iPad
3. Tap the `.mcaddon` → Share → Minecraft

### World settings checklist

Before tapping **Create**, verify:

- [ ] **Holiday Creator Features** — ON
- [ ] **Beta APIs** — ON  
- [ ] **Resource pack** — Brindal & Grayson Cow World — Active
- [ ] **Behavior pack** — Brindal & Grayson Cow World BP — Active
- [ ] This is a **NEW** world (not editing an old one)

### Verify it works

1. Open chat → type `!moo` → a cow appears
2. Type `!b` → Brindal's brown cow appears
3. Type `!g` → Grayson's gray cow appears
4. Find a zombie — it should look and sound like a cow

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Packs don't appear | Re-import `.mcaddon`; force-quit and reopen Minecraft |
| Missing textures (checkerboard) | Activate **both** resource AND behavior packs |
| `!moo` does nothing | **Beta APIs** must be ON in this world's settings |
| Mobs aren't cows | **Holiday Creator Features** ON; create a **new** world |
| Brindal/Grayson cows missing | They won't auto-spawn everywhere — use `!b` or `/summon bgcow:brindal_cow` |
| "Failed to import" | Update Minecraft to 1.21.0+; free up storage |
| Lag with `!mega` or `!rain` | Normal on older iPads — stick to `!moo` and `!party` |
